## 简介

`Saidmin Vue` 5.x

## 安装使用

- 获取代码

```
git clone https://github.com/saithink/saiadmin-vue.git
```

或

```
git clone https://gitee.com/appsai/saiadmin-vue.git
```

或

```
git clone https://gitcode.com/saigroup/saiadmin-vue.git
```

- 安装依赖

```
cd saiadmin-vue && yarn install
```

- 运行

```
yarn dev
```

- 打包

```
yarn build
```
